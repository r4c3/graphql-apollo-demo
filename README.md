## Running Locally

1. Clone the GitHub Project
   ```
   $ git clone https://github.com/r4c3/graphql-apollo-demo.git
   ```
2. Navigate to the client folder, download packages, and run
   ```
   $ cd graphql-apollo-demo
   $ yarn i
   $ yarn start
   ```
